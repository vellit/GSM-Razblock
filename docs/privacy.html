# Политика конфиденциальности приложения «2G Разблок» (ru.vellit.gsm2g) Последнее обновление: 18 января 2026 ## Кто мы Разработчик: Vellit Контакт для поддержки: gsmrazblock@gmail.com ## Какие данные обрабатываются - **Локальные данные на устройстве:** список выбранных пользователем приложений (белый список) и настройки напоминаний. Хранятся только на устройстве, не отправляются на сервер. - **Системные разрешения:** - Доступ к статистике использования (PACKAGE_USAGE_STATS) — используется для определения активного приложения, чтобы показать напоминание. - Уведомления (POST_NOTIFICATIONS) — используются для показа напоминаний. - **Не собираем и не передаём:** персональные данные, учётные записи, историю звонков, местоположение, содержимое файлов, сеть и т.д. ## Цели использования данных - Показывать напоминание вернуться к обычному режиму сети, когда активировано приложение из белого списка. - Открывать системное меню Phone info по кнопке. ## Хранение и передача - Все данные хранятся локально на устройстве и удаляются при удалении приложения или очистке его данных. - Данные не передаются третьим лицам и не отправляются на внешние серверы. ## Безопасность - Приложение не использует шифрование для локально хранимых данных, так как данные не покидают устройство. - Рекомендуется не использовать приложение на взломанных/рутованных устройствах. ## Права пользователя - Вы можете в любой момент отозвать разрешения в настройках системы (Usage Access, уведомления). - Вы можете удалить или изменить белый список в настройках приложения. - Вы можете удалить приложение, чтобы стереть все локальные данные. ## Изменения в политике Обновления политики будут публиковаться по тому же URL. Дата последнего обновления указана выше. ## Контакты По вопросам приватности: gsmrazblock@gmail.com
